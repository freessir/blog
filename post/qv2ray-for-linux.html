米白云 Qv2ray 软件使用教程

发表于2021-07-02|更新于2021-07-02
|字数总计:1k|阅读时长:3分钟|阅读量:

介绍

使用 Qt 编写的 v2ray 跨平台 GUI，支持MacOS、Windows和Linux，支持连接导入和编辑，中英文切换。
因为是 GUI 可以进行可视化操作，无需借助终端，上手难度比 Clash 低。

Linux 系统其他软件：

  • Qv2ray
  • Clash

下载安装

下面两个方式选其一下载：

  • GitHub 官网
  • 阿里云镜像站

设置权限

右键下载后的文件，点击 属性 —> 权限，勾选 允许执行文件。

然后即可双击打开，打开后软件界面如下图所示。

下载内核

Qv2ray 运行依赖 V2ray 内核，但是 Qv2ray 默认不包含 V2ray 内核（是不是听着很扯）。接下来你要根据自己的操作系统下载 V2ray 内核。

  • V2ray 内核下载

判断哪个软件版本适合我？
一般只要根据自己使用的几位操作系统下载：64 位操作系统 选择下载 v2ray-linux-64.zip；32 位操作系统 选择下载 v2ray-linux-32.zip；如果你的处理器是 arm 构架，根据几位操作系统，下载对应文件。

这里建议将下载的 .zip 压缩包放置在 Qv2ray 同目录下，然后解压。

加载内核

点击 Qv2ray 软件上方 首选项，位置下图所示：

点击 内核设置，这里需要填写两个地方：

  • V2Ray 资源目录：选择刚才解压出来的目录（文件夹）
  • V2Ray 核心可执行文件路径：选择刚才解压出来的目录下载 V2ray 文件。

这两者就差个 /V2ray 。

然后点击下面 检查 V2Ray 核心设置 按钮，当提示 V2Ray 路径配置检查通过 就表示内核加载成功，然后右下角 OK。

导入订阅

打开 米白云首页，找到 ⚡️便捷导入 ，位置在电脑端的右下角、手机端的最下方。点击 SSR订阅链接 —> 复制 V2Ray 订阅链接。

点击 Qv2ray 上方 分组

分组名称 修改为 米白云 。然后切换到 **订阅设置**，勾选 **此分组是一个订阅**，订阅地址 粘贴刚才复制的米白云 V2ray 订阅链接，然后点击下方的 **更新订阅** 按钮，右下角 **OK**。

使用 Qv2ray

打开 Qv2ray 软件，在左侧空白区域顶部出现 米白云，双击即可出现节点列表，选择一个节点（不要前两个），然后点右上角 ▶ 开启。

浏览器打开谷歌：google.com，没有问题的话就能正常打开，配置成功。

（可选）安装SSR插件

Qv2ray 软件除了支持 v2ray 节点外，还支持 ssr 节点。通过安装 ssr 插件以启用 ssr 节点。

下载 SSR 插件

到 releases page 或者 阿里云镜像站 下载对应的 .so 插件，记住插件的放置路径。

加载 SSR 插件

点击 Qv2ray 软件上方 插件，位置如下图：

点击左下方 打开本地插件目录，将刚才下载的插件拖到本地插件目录。

执行 kill 32054 即可彻底结束 Q2v2ray 进程，现在再重新打开即可看到插件已经加载成功。

参考上面 导入订阅 的方法， 复制 SSR 订阅链接 并导入 SSR 订阅即可。

一般问题

每次都要打开米白云网页才能使用？

首次配置后无需再次配置，也不需要再打开米白云网页，下次打开 V2RayN 软件即可使用。

更换套餐后无法使用？

如果你更换了套餐，更新订阅即可使用。方法：点击 Qv2ray 软件上方 分组，切换到 订阅设置，点击下方 更新订阅 按钮。

不使用时如何关闭？

在 Qv2ray 点击右上角 ，然后关闭 Qv2ray 软件。Linux版 Qv2ray 关闭窗口即退出，如要继续使用请最小化它。
