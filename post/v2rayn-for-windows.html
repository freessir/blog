米白云 V2RayN 软件使用教程

更新于 2021-06-29 阅读次数： Valine：
本文字数： 806 阅读时长 ≈ 1 分钟

软件介绍

V2RayN 是一个基于 V2Ray 内核的 Windows 客户端。速度和稳定性都不错，目前保持活跃更新中。

米白云 提供 SS、SSR 和 vmess 三种类型节点，使用 V2RayN 软件只能获取 vmess 类型节点，如需所有类型节点，请使用： Clash for Windows

适用于 Windows 系统的软件及其教程：

  • V2rayN
  • SSTap
  • Netch
  • ShadowrocksR
  • Clash for Windows

下载软件

下面列表方式中选其一下载：

  • 蓝奏网盘
  • GitHub 官网
  • 阿里云镜像站

安装软件

将下载好的压缩包右键 解压到“v2rayN-Core\“（E）

打开解压后到文件夹 v2rayN-Core，找到带有图标的 v2rayN 并双击。

v2rayN为绿色开源软件，无需安装，打开即用。

导入订阅

打开 米白云首页，找到 ⚡️便捷导入，位置在电脑端的右下角、手机端的最下方。点击 SSR订阅链接 => 复制 V2Ray 订阅链接。

打开 V2RayN 软件，点击左上方 订阅 => 订阅设置。

点击左下方 添加(A)，备注 填写 米白云，地址（url） 粘贴刚才复制的 米白云v2ray链接。最后右下角 确定。

重新点击右上角 订阅，然后 更新订阅。可以看到节点列表即表示导入成功。

使用 V2RayN

关闭 V2RayN 软件窗口，找到在桌面右下角处的图标右键，点击 服务器，选择一个节点。

再次图标右键，点击 系统代理，选择 自动配置系统代理。此举可以实现只有非大陆地区走代理，聊微信、QQ，浏览百度等国内应用不走代理。

浏览器打开谷歌：google.com，没有问题的话就能正常打开，配置成功。

一般问题

不使用如何关闭

在桌面有下角图标右键，选择 系统代理，选择 清除系统代理 即可。或者直接右键退出软件。

无需打开米白云网页

首次配置后无需再次配置，也不需要再打开米白云网页，下次打开 V2RayN 软件即可使用。

更换套餐后无法使用

如果你更换了套餐，更新订阅即可使用。方法：在桌面有下角图标右键 => 点击 更新订阅。
