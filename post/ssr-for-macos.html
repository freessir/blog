介绍

ShadowsocksX-NG-R8 为 MacOS系统 上的ssr软件。目前作者已不再提供更新维护，软件在 MacOS11 新系统上会有莫名其妙的问题，例如：节点不通、网页打不开。遇到此类问题并一直无法解决的用户建议转用 ClashX。

MacOS 系统其他软件：

  • ClashX
  • V2rayU
  • SSR
  • CFW

下载安装

下面列表方式选其一下载：

  • 蓝奏网盘​
  • GitHub 官网​

此软件无需安装，双击解压 zip压缩文件 后出现一个 ShadowsocksX-NG-R 文件，这就是应用软件。双击解压出来的 ShadowsocksX-NG-R 软件，首次打开会询问你 这是一个来自互联网的软件，是否打开？，点 打开，然后输入开机密码。

导入订阅

打开 米白云首页，找到 ⚡️便捷导入 ，位置在电脑端的右下角、手机端的最下方。点击 SSR订阅链接 -> 复制 SSR 订阅链接。

在菜单栏的右上方出现小飞机图标，依次点击 服务器订阅 -> ⚙ 订阅设置。

点击左下方 ，在订阅处粘贴刚才复制过的 米白云SSR链接 ，最后点右下角 更新。

如果提示更新失败，请依次点击 服务器订阅 -> 更新订阅（不通过代理），即可更新成功。

默认的节点是一条米白云的通知信息，不是有效节点。按照下图所示更换一个有效节点。节点名后的数字为延迟，有延迟的节点均为有效节点。
