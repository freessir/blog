介绍

Netch 是一款 Windows 平台的开源游戏加速工具，和SSTap一样可以使用全局代理加速游戏。不同的是 Netch 软件目前仍在维护更新中。

Netch 适合游戏，如果你仅是为了日常浏览网页，请使用 Clash for Windows。

适用于 Windows 系统的软件及其教程：

  • V2rayN
  • SSTap
  • Netch
  • ShadowrocksR
  • Clash for Windows

下载软件

下面方式选其一下载：

  • 蓝奏网盘
  • GitHub 官网
  • 阿里云镜像站

安装软件

Netch 是一款绿色软件，无需安装，即点即用。

打开一直报错怎么办？

Netch 软件的运行依赖 .NET4.8+，只需要点击 👉 这里 即可从微软官网下载 .NET 并安装即可。下载前请自行分辨自己是多少位的操作系统，默认下载 64 位。

导入订阅

打开米白云首页，找到 ⚡️便捷导入，位置在电脑端的右下角、手机端的最下方。点击 SSR订阅链接 ==> 复制 SSR 订阅链接。

Netch 软件支持 SS、SSR和 vmess 三种类型节点。这里以导入 SSR 节点为例，同样方法你也可以导入 SS 和 V2ray 节点。

打开Netch，点击上方 订阅 ==> 管理订阅链接。

备注 填写 米白云，链接 粘贴在米白云复制的 SSR 订阅链接，然后右下角 添加 并关闭窗口。

添加了 SSR 订阅链接 后，当你还需要添加 SS 和 V2ray 订阅，，可以备注为 米白云 SS、米白云 Vmell。

点击软件上方 订阅 ==> 从订阅链接更新服务器。更新成功后在桌面右下方提示从米白云更新了多少个节点。

使用 Netch

参照下图：服务器 选择一个节点，然后点击右侧 ⚡️ 图标测试该节点是否可用；选择 模式，这里选择 [3] Bypass LAN and China；最后点击右下角 启动。

Netch适合游戏代理加速，需要日常网页浏览、看视频等请使用 Clash 软件 或 SSR 软件 。

什么是模式？
模式可以理解为要代理加速什么应用。Netch内置多个游戏和应用的模式，如你需要玩 美服英雄联盟，就在模式里选择 [1] League of Legends（US）。

如果你的电脑没有安装过虚拟网卡，启动时会让你安装，请点击 **安装(I)**。

浏览器打开谷歌：google.com，没有问题的话就能正常打开，配置成功。

一般问题

Win7 系统无法打开

win7 系统需要安装补丁：kb4503292。请点击 👉 这里 从微软官网下载对应你操作系统的补丁，点击右边的 Download 即可下载。在弹出的页面中点击链接下载并安装。安装成功后你的电脑需要重启以完成补丁的安装。安装成功后 Netch 软件即可打开。

更换套餐无法使用

如果你更换了套餐，更新订阅即可使用。方法：点击 Netch 软件上方 订阅 ==> 从订阅链接更新服务器。

创建自定义模式

点击上面 模式 ==> 创建进程模式，点击左下角 扫描，选择想要代理加速的软件的安装目录。如我要代理火狐浏览器，我选择的路径：C:\Program Files（x86）\Mozilla Firefox，然后点击下方 选择文件夹，软件自动加载所有的 exe文件。在上方 备注 处填写软件名，如这里是 Firefox，最后右下角 保存。现在在模式里能看到 Firefox 了。
