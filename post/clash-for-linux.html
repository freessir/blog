米白云 Linux版 Clash 软件通用教程

发表于2021-07-02|更新于2021-07-02
|字数总计:553|阅读时长:2分钟|阅读量:

简介

此处为Linux通用教程，教程演示的是 Ubuntu 18 系统。本教程默认使用 Linux 的用户均具备一定的基础知识和动手能力，故其他系统不再另写教程，请自行参考此篇教程。

必备工具

提前安装必要的工具，避免后续安装过程出现错误。

#CentOS系统

#Ubuntu、Debian系统

在当前用户目录下创建 clash 文件夹，用户存放 clash 软件和配置文件，便于管理

下载软件

下载 clash 软件

上面命令模式下载的是 Intel X64 操作系统 适用的软件。如果你是其他操作系统请更换对应的软件。

  • Github官网
  • 阿里云镜像站

下载 Country.mmdb 数据库

下载 Country.mmdb 数据库文件供 clash 分流使用。

导入订阅

打开 米白云首页，找到 ⚡️便捷导入 ，位置在电脑端的右下角、手机端的最下方，点击 复制Clash 订阅链接。将复制的订阅链接替换为下方第二行命令中的 clash订阅。不要忘了双引号和最后的 &new=1 。

启动 Clash

使用Clash

浏览器访问：clash.razord.top，进入网页版 clash 设置页面，可以进行节点切换、规则更改等操作。

打开系统设置界面，Ubuntu 18可以在桌面空白处 右键 —> 设置 打开。打开 网络代理。

网络代理改成 手动，填写

  • HTTP 代理：127.0.0.1 7890
  • HTTPS 代理：127.0.0.1 7890
  • Socks 主机：127.0.0.1 7891

最后浏览器打开 google.com，成功打开即表示配置成功。
