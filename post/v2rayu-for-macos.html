米白云 V2rayU 软件使用教程

更新于 2021-06-30 阅读次数： Valine：
本文字数： 644 阅读时长 ≈ 1 分钟

介绍

V2rayU，基于 v2ray 核心的mac版客户端，小巧强大。

MacOS 系统下其他软件下载及教程：

  • ClashX
  • V2rayU
  • SSR
  • CFW

下载安装

下面列表方式选其一下载：

  • 蓝奏网盘
  • GitHub 官网
  • 阿里云镜像站

将 V2rayU 图标移动到 Applications 完成安装。

通过启动台查找 V2rayU 启动。首次使用询问 互联网下载的App是否确定打开，点击 打开。

输入开机密码后，允许 V2rayU 接受传入网络链接。

导入订阅

打开 米白云首页，找到 ⚡️便捷导入，位置在电脑端的右下角、手机端的最下方。点击 SSR订阅链接 ==> 复制 V2Ray 订阅链接。

在桌面上面菜单栏右侧点击 V2rayU 图标，点击 订阅设置。

地址 粘贴刚才复制的 米白云 V2ray 订阅链接，备注 填写 米白云。然后 添加，再点右下角 更新。

出现 订阅更新日志，显示订阅导入成功。

使用 V2rayU

点击 服务器列表，选择一个节点。

选择节点后，V2rayU 自动开启代理。或者确保顶部 V2ray-core 为 On，Turn V2ray-core 为 Off。

浏览器打开谷歌：google.com，没有问题的话就能正常打开，配置成功。

一般问题

不使用时如果关闭

打开 V2rayU 软件，在顶部处点击 Turn V2ray-core Off，V2ray-core 变成 Off 即关闭成功。下次使用时点击 Turn V2ray-core On 开启使用。或者直接点击最下方退出，下次使用重新打开。

更换套餐后无法使用

如果你更换了套餐，更新订阅即可使用。方法：在桌面有下角图标右键，点击 订阅 ==> 更新。
