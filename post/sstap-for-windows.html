米白云 SSTap 软件使用教程

发表于2021-06-29|更新于2021-07-02
|字数总计:710|阅读时长:2分钟|阅读量:

软件介绍

一款适合游戏加速的Windows软件。稳定好用，但作者早已不再维护，好在老版本依旧好用。

米白云 提供 SS、SSR 和 vmess 三种类型节点，使用 SSTap 软件只能获取 SS 和 SSR 类型节点，如需所有类型节点，请使用： Clash for Windows

Windows 系统其他软件：

  • V2rayN
  • SSTap
  • Netch
  • ShadowrocksR
  • Clash for Windows

软件下载

下面的下载方式选其一下载：

  • 蓝奏网盘
  • GitHub 官网
  • 阿里云镜像站

软件安装

下载解压后进入文件夹，得到 SSTap 安装程序。

双击安装，一路 下一步、同意 即可。

如果想要更改安装路径，请点击 **浏览(B)…**，然后点击右下角 **安装(I)**。

最后点击 完成。

第一次使用弹出 警告，点 接受。

导入订阅##

打开 米白云首页，找到 ⚡️便捷导入 ，位置在电脑端的右下角、手机端的最下方。点击 SSR订阅链接 -> 复制 SSR 订阅链接。

打开 SSTap 软件，点击右下角 ，下图所示：

然后依次点击 *SSR订阅 -> SSR订阅管理…

URL 粘贴刚才复制的 SSR 订阅链接，点击 添加 后关闭。

使用 SSTap

关闭上面的窗口后，软件会自动更新订阅。当提示订阅成功后，点击下方所示位置的下拉框，选择一个节点（不要选前三个）。

选择后点击最右侧 ⚡ ️按钮，会自动测试选择的节点连接情况。为了降低游戏时延迟，游戏商一般都通过 UDP 进行连接。米白云只有部分节点支持 UDP，请自行测试。

如果没有问题，点击下方的 连接 按钮。

允许通过防火墙。

浏览器打开谷歌：google.com，没有问题的话就能正常打开，配置成功。

一般问题

如何做到国内不走代理

打开 SSTap 软件，点击 模式 选择 仅网页浏览器(跳过中国站点)。

软件模式没有我要的游戏

模式中选择游戏就能加速对应游戏。SSTap 默认内置很少的游戏模式，如果想要更多游戏：

  • 先关闭退出SSTap，找到SSTap的安装目录并打开该目录下的rules文件夹
  • 下载：规则压缩包，
  • 将解压得到 rules 文件夹，将此文件夹下所有的文件复制到 SSTap 软件安装目录下的 rules 文件夹里，跳过相同的文件
  • 重新打开 SSTap 软件即可在模式里看到更多的游戏

Win10 开启代理后 CPU 占用很高

这是由 Windows 10 的一个网络检测服务所致，SSTap 虚拟网卡创建的连接被误认为是受限网络，从而触发了检测。无需理会，数分钟以后会恢复正常。
