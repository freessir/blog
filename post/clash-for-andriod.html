米白云 Clash for Andriod 软件使用教程

更新于 2021-06-30 阅读次数： Valine：
本文字数： 662 阅读时长 ≈ 1 分钟

简介

Clash for Andriod 是使用在安卓系统的 clash 软件，稳定可靠，目前仍在活跃更新中。

安卓系统下其他软件下载及安装：

  • Clash for Andriod
  • SSRRAY
  • V2rayNG

下载安装

下面列表方式中选其一下载：

  • 蓝奏网盘
  • GitHub 官网
  • 阿里云镜像站
  • Play Store

安卓软件安装过于简单，不做介绍。

导入订阅

导入订阅前请确保：

  • 购买了米白云套餐
  • Clash软件已打开

打开 米白云首页，找到 ⚡️便捷导入，位置在电脑端的右下角、手机端的最下方。点击 一键导入 Clash 配置。

自动跳转到 Clash 软件，修改名称为 米白云；修改 自动更新(分钟) 为 60，然后点击右上角的 保存 图标。

点击 配置

点选 米白云，然后左上角 <— 返回。

使用 Clash

现在看到 配置 显示 米白云已激活；点击 已停止 开启Clash，现在软件显示运行中，如下图所示。

接下来浏览器打开谷歌首页：google.com，没有问题的话就能正常打开，配置成功。

一般问题

怎么开启全局？

依次点击 设置 —> 覆写 —> 模式，选择 全局模式，重新回到 Clash 软件首页，点击 代理，即可看到 GLOBAL，全局需要自行指定节点。如非必需，请不要开启全局代理。

如何更换节点

打开 Clash 软件，点击 代理，选择一个节点。如没有特殊要求，请选择 负载均衡，可以提升上网体验。

不使用时如何关闭

打开 Clash 软件，点击 运行中，当它变成 已停止 即关闭。或者在手机后台直接清除 Clash 软件。

更换套餐后无法使用

如果你更换了套餐，更新订阅即可使用。方法：打开 Clash 软件，点击 配置，点击 米白云 后三个点，然后点击 更新。
