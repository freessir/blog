米白云 SSRRAY 软件使用教程

更新于 2021-06-30 阅读次数： Valine：
本文字数： 727 阅读时长 ≈ 1 分钟

简介

SSR全称ShadowsocksR，常被叫做 “酸酸乳“、“小飞机“、“纸飞机“ 等。本页提供的是安卓系统的 SSR 教程。SSRRAY 软件为目前 SSR 在安卓系统最好软件，推荐使用这个。

安卓系统下其他软件下载及安装：

  • Clash for Andriod
  • SSRRAY
  • V2rayNG

安装下载

下列方式选其一下载：

  • 蓝奏网盘
  • GitHub 官网
  • 阿里云镜像站

安装过于简单，略过。

导入订阅

打开 米白云首页，找到 ⚡️便捷导入，位置在电脑端的右下角、手机端的最下方。点击 SSR订阅链接 ==> 复制 SSR 订阅链接。

打开左上角的 ShadowsocksR

左滑删除默认配置文件

点击右下角 + 号 ==> 管理SSR|V2Ray订阅

点击右上角 + 号

将 群组名 改为 米白云，粘贴刚才复制的米白云 SSR 订阅，然后点右下角 确定。

当出现 米白云，表示订阅导入成功。点击左上角的 X 关闭当前页面。

使用SSR

现在我们能看到节点出现，任选一个节点。

下滑到 功能设置，将路由设置为 绕过局域网及中国大陆地址，然后点击右上角到开关。

第一次使用还会询问是否允许连接，点 确定。

在右上角会提示连接成功并显示网速信息。浏览器打开谷歌首页：google.com，没有问题的话就能正常打开，配置成功。

常见问题

如何更换节点

点击左上角 ShadowsocksR，出现节点列表，选择你想要到节点。如出现节点无法使用的情况，也可以通过此操作更换其他节点。

不使用时如何关闭

打开 SSRRAY 软件，点击右上角开关关闭。或者直接后台清除软件。

更换套餐后无法使用

如果你更换了套餐，更新订阅即可使用。方法：打开 SSRRAY软件，点击左上角 ShadowsocksR —> 点击右下角 + —> 管理 SSR|V2Ray 订阅 —> 米白云后面的三点 —> 更新。
