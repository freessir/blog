米白云 SSR 软件使用教程

更新于 2021-06-29 阅读次数： Valine：
本文字数： 962 阅读时长 ≈ 1 分钟

简介

ShadowsocksR 简写 SSR，常被叫做 “酸酸乳“、“小飞机“、“纸飞机“ 等。目前此软件已经停止更新，仅有志愿者提供维护。下面提供的就是志愿者维护的版本。

下载安装

下载列表选其一下载：

  • 蓝奏网盘 ✅
  • Github官网
  • 阿里云镜像站 ✅
  • 微软云加速（download.fastgit.org）

为了便于管理，右键下载的压缩包，选择 解压到当前文件夹(X)，解压会在同目录下出现 ShadowsocksR 文件夹。

进入文件夹后双击小飞机运行。这是一款免安装的绿色开源软件，使用前最好关闭国内的电脑管家如：360、腾讯电脑管家、百度电脑管家。

第一次使用点 否，软件自动打开订阅设置窗口。

导入订阅

打开 米白云首页，找到 ⚡️便捷导入，位置在电脑端的右下角、手机端的最下方。点击 SSR订阅链接 => 复制 SSR 订阅链接。

打开订阅设置窗口，或者在桌面右下角小飞机图标上右键，依次点击 订阅 => 设置。

点击左下角 添加，然后删除右上角 订阅地址，右键粘贴就能填入刚才复制的 米白云SSR订阅链接，再点击 更新当前订阅，当订阅备注变为 米白云 表示更新成功，最后关闭窗口。

如果无法更新成功，请把更新订阅使用的代理类型由 自动 改为 直连，改动位置下图所示。

开启SSR

如果对节点ip无要求，开启负载均衡可以帮助你获得更好的上网体验。在桌面右下角小飞机图标右键，依次 服务器 => 负载均衡，点击即可开启负载均衡。

再次桌面由下角小飞机图标右键，依次 系统代理 => PAC，现在你可以尝试上外网了。如果你不想使用，可以将系统代理改为不修改。

PAC模式允许你只在海外网站走代理，国内如微信、QQ、百度等不走代理。

浏览器打开谷歌：google.com，没有问题的话就能正常打开，配置成功。

一般问题

浏览器无法上网

确定按照教程所有都没问题，浏览器仍然无法上网，请为你的浏览器：安装 SwitchyOmega。

打开米白云才能使用

首次配置后无需再次配置，也不需要再打开米白云网页，下次打开 SSR 软件即可使用。

更换套餐后无法使用

如果你更换了套餐，更新订阅即可使用。方法：在桌面有下角图标右键，点击 订阅 => 更新。

不使用如何关闭

在桌面有下角图标右键，选择 系统代理，选择 直连 即可。或者直接右键退出软件。
