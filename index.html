米白云 SSR 软件使用教程

更新于 2021-06-28 Valine：
本文字数： 916 阅读时长 ≈ 1 分钟

简介

ShadowsocksR 简写 SSR，常被叫做 “酸酸乳“、“小飞机“、“纸飞机“ 等。目前此软件已经停止更新，仅有志愿者提供维护。下面提供的就是志愿者维护的版本。

下载安装

下载列表选其一下载：

  • 蓝奏网盘 ✅
  • Github官网
  • 阿里云镜像站 ✅
  • 微软云加速（download.fastgit.org）

为了便于管理，右键下载的压缩包，选择 解压到当前文件夹(X)，解压会在同目录下出现 ShadowsocksR 文件夹。

进入文件夹后双击小飞机运行。这是一款免安装的绿色开源软件，使用前最好关闭国内的电脑管家如：360、腾讯电脑管家、百度电脑管家。

第一次使用点 否，软件自动打开订阅设置窗口。

导入订阅

打开 米白云首页，找到 ⚡️便捷导入，位置在电脑端的右下角、手机端的最下方。点击 SSR订阅链接 => 复制 SSR 订阅链接。

打开订阅设置窗口，或者在桌面右下角小飞机图标上右键，依次点击 订阅 => 设置。

点击左下角 添加，然后删除右上角 订阅地址，右键粘贴就能填入刚才复制的 米白云SSR订阅链接，再点击 更新当前订阅，当订阅备注变为 米白云 表示更新成功，最后关闭窗口。

如果无法更新成功，请把更新订阅使用的代理类型由 自动 改为 直连，改动位置下图所示。

开启SSR

如果对节点ip无要求，开启负载均衡可以帮助你获得更好的上网体验。在桌面右下角小飞机图标右键，依次 服务器 => 负载均衡，点击即可开启负载均衡。

再次桌面由下角小飞机图标右键，依次 系统代理 => PAC，现在你可以尝试上外网了。如果你不想使用，可以将系统代理改为不修改。

PAC模式允许你只在海外网站走代理，国内如微信、QQ、百度等不走代理。

浏览器打开谷歌：google.com，没有问题的话就能正常打开，配置成功。

一般问题

浏览器无法上网

确定按照教程所有都没问题，浏览器仍然无法上网，请为你的浏览器：安装 SwitchyOmega。

打开米白云才能使用

首次配置后无需再次配置，也不需要再打开米白云网页，下次打开 SSR 软件即可使用。

更换套餐后无法使用

如果你更换了套餐，更新订阅即可使用。方法：在桌面有下角图标右键，点击 订阅 => 更新。

不使用如何关闭

在桌面有下角图标右键，选择 系统代理，选择 直连 即可。或者直接右键退出软件。

米白云 Clash for Windows 软件使用教程

更新于 2021-06-28 Valine：
本文字数： 1.5k 阅读时长 ≈ 1 分钟

简介

Clash for Windows是基于 Clash 内核 在Windows系统下的一款优秀的开源软件，目前仍保持着活跃的更新，安全性和稳定性都有可靠保证。

软件下载

下面列出的下载地址任选其一：

  • 蓝奏网盘​
  • Github 官网​
  • 阿里云镜像站​
  • 微软云加速（download.fastgit.org）

请按照下面的图文指示进行安装。已经安装了的请跳过到第三步骤。

导入订阅

导入订阅前请确保：

  • 购买了米白云套餐​
  • Clash软件已打开

打开米白云首页，找到 ⚡️便捷导入，位置在电脑端的右下角、手机端的最下方。点击 一键导入 Clash 配置。

浏览器弹出窗口询问是否打开Clash软件，点击 **选择应用程序(A)**。不同浏览器弹窗内容不太一样，但意思都差不多，只要不点取消就行。

在Clash上切换到Profiles，列表里一般会有两个配置文件，点击第二个配置文件的文件名来选择第二个配置文件。Clash如何配置中文？

点击后面的 ⚙ 按钮，修改 Name 为米白云，和 Update Interval(hour) 为1，此项修改配置文件名为 米白云 便于识别；开启每小时自动更新订阅。请务必确保设置自动更新订阅。

开启Clash

切换到 General，打开 System Proxy 开关。Clash 默认只走海外流量，开着Clash并不会影响你浏览百度、聊微信QQ。

接下来浏览器打开谷歌首页：google.com，没有问题的话就能正常打开，配置成功。

常见问题

如何更换节点

点击 Proxies，点击 节点选择，在出现的节点列表中选择你要的节点即可。如果对节点无要求，请选择 负载均衡，软件会动态匹配最低延迟的节点。

误区：延迟并不能代表网速。

不使用时如何关闭

打开Clash for Windows，切换到 General，关闭 System Proxy 开关即可。或者直接右键 Quit
退出软件。

浏览器无法上网

确定按照教程所有都没问题，浏览器仍然无法上网，请为你的浏览器：安装 SwitchyOmega 插件。

Clash 加速其他设备

请参考：此处。

需要打开网页才能使用

首次配置后无需再次配置，也不需要再打开米白云网页，下次打开 Clash 软件即可使用。

更换套餐后无法使用

如果你更换了套餐，更新订阅即可使用。方法：在桌面有下角图标右键，点击 更新订阅。

如何配置中文

Clash 软件官方版默认没有中文，但可以通过手动导入 中文语言包 实现。语言包与 Clash 版本对应，并非通用，请根据自己使用的 Clash 版本选择对应的语言包。

  • 0.15.3 版本中文语言包​
  • ​0.15.4 版本中文语言包​
  • 其他版本语言包：TG 群​

如果你正在使用的版本号不在上述列表里，请升级你的 Clash 到对应版本。打开 Clash 点击 General，即可看到版本号。

首先退出 Clash，在桌面 Clash 图标右键选择 打开文件位置(I)。如果你清楚 Clash 的安装位置，可以直接打开。
在安装目录里打开 resources 文件夹，删除 app.asar 文件，或者将其重命名为 bak.app.asar。将下载的中文语言包解压，将其中的 app.asar 文件复制到 resources 文件夹里。

重启 clash 即可看到中文。

MacOS 版同 Windows 方法一样，只是 app.asar 文件路径不一样。(注意不是 ClashX)

MacOS 修改方法大致：

打开 访达 —> 应用程序 —> 找到 Clash for Windows —> 右键 Clash for Windows —> 点击 显示包内容 —> Contents —> Resources。替换里面的 app.asar 文件为 语言包里的 app.asar 文件。

计划任务：crontab 对比 anacron

更新于 2021-06-27 分类于 linux study Valine：
本文字数： 3.2k 阅读时长 ≈ 3 分钟

cron 用于周期性执行定时任务，它的 daemon 叫做 crond，常驻内存。crontab 是用于编辑管理 cron 的指令。同样的，anacron 是指令，anacrontab 是其配置文件。

作为系统服务，cron 可以由 systemctl 管理，能够以分钟精度执行任务。而 anacron 仅是一个指令，需要 cron 调用才能运行。cron 认为主机会 24 小时不间断运行。 anacron 考虑到宕机等实际问题，认为主机不可能不出意外一直连续运行下去，因此会定时执行因意外导致无法被 cron 执行的任务。anacron 可以算是 cron 的补充和完善。
